## 相关性矩阵的构建

计算出三个相关系数后，需要构建3×3相关性矩阵用于蒙特卡洛模拟：

```
         W_S    W_r/W_L    W_FX
W_S      1      ρ_SL       ρ_SFX
W_r/W_L  ρ_SL   1          ρ_LFX
W_FX     ρ_SFX  ρ_LFX      1
```

其中：
- W_S：驱动quanto股指S(t)的布朗运动
- W_r（Hull-White）或W_L（LMM）：驱动利率的布朗运动
- W_FX：驱动汇率的布朗运动（虽不直接出现在payoff，但影响quanto adjustment）

**Cholesky分解**：
将相关性矩阵进行Cholesky分解，得到下三角矩阵L，使得：
- ρ = L × L^T
- 在模拟中：(W_S, W_r, W_FX)^T = L × (Z_1, Z_2, Z_3)^T
- 其中Z_1, Z_2, Z_3是独立的标准正态随机变量

## 2.2 SX5E Index Data

We use the **EURO STOXX 50 Total Return Index (SX5T)** as our equity 
underlying, which assumes dividend reinvestment. This choice offers 
several advantages:

1. **Simplified modeling**: The dividend yield q can be set to 0, 
   as dividends are already incorporated in the index level.

2. **Accurate volatility estimation**: Historical volatility computed 
   from total return index is not distorted by discrete dividend payments.

3. **Market convention**: Total return indices are standard for 
   long-dated equity derivatives pricing.
